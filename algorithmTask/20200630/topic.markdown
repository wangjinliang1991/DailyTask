## 13. 罗马数字转整数
罗马数字包含以下七种字符: I， V， X， L，C，D 和 M。
同上道题...

给定一个罗马数字，将其转换成整数。输入确保在 1 到 3999 的范围内。
示例 1:
```
输入: "III"
输出: 3
```

示例 2:
```
输入: "IV"
输出: 4
```

示例 3:
```
输入: "IX"
输出: 9
```

示例 4:
```
输入: "LVIII"
输出: 58
解释: L = 50, V= 5, III = 3.
```

示例 5:
```
输入: "MCMXCIV"
输出: 1994
解释: M = 1000, CM = 900, XC = 90, IV = 4.
```

### 题解
当小值在大值的左边，则减小值
当小值在大的右边，加小值
右值永远为正，最后一位必为正

代码实现：往右多看一位，对比当前位和后一位的大小关系，从而确定是
加法还是减法，没有下一位时，加法即可
